This project is for COS website that headless CMS ( WordPress + React).
This project made by `npx create-react-app my-app`

## start

`yarn start`

## Library

- typescript
- react-router-dom
- styled-components
- react-media
- react-player
- react-burger-menu
- styled-icons
- react-svgmt

### wordpress plugin

- Advanced Custom Fields
- Custom Post Type UI
- WP REST API Menus
